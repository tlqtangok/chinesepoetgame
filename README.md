# chinesepoetgame
儿歌诗词填词游戏

## 功能特点

- 🎮 适合5岁小朋友的可爱风格诗词填字游戏
- 🔊 点击任意汉字可朗读普通话发音
- 🎯 多音字智能发音，根据语境选择正确读音
- ↩️ 支持撤销上一次拖拽操作
- 📱 支持触摸拖拽，适配移动设备
- 🍎 禁用Safari双击放大，优化移动体验
- 📐 响应式设计，自适应屏幕尺寸

## 游戏玩法

1. 游戏会显示两句儿歌诗句，部分汉字会被移除
2. 将下方的汉字拖拽到上方对应的空格中
3. 点击任意汉字可以听到普通话发音
4. 如果拖错了，可以点击右上角的撤销按钮回退一步
5. 全部填对后会显示成功提示，填错会自动重置重试

## 技术实现

- 纯HTML/CSS/JavaScript实现，无需依赖
- 使用Web Speech API实现汉字发音
- 支持桌面拖拽和移动端触摸操作

## 更新日志

### 多音字发音优化

识别并修正了儿歌中的多音字发音，确保根据语境正确朗读：

| 诗句 | 多音字 | 正确读音 | 说明 |
|------|--------|----------|------|
| 背上小书包 | 背 | bēi | 背负的意思，非 bèi（背部） |
| 数星眨眼睛 | 数 | shǔ | 动词，数数的意思 |
| 铅笔写数字 | 数 | shù | 名词，数字的意思 |
| 筷子夹豆角 | 角 | jiǎo | 豆角，非 jué（角色） |
| 关灯睡觉觉 | 觉 | jiào | 睡觉，非 jué（感觉） |

其他多音字如"着"、"藏"、"空"在当前语境下TTS引擎能正确识别，无需特殊处理。
